# MyPDF-Manager 技能

这是一个 OpenClaw 技能，用于 PDF 文件处理。

## 快速开始

1. 将此目录放置在 `~/.openclaw/workspace/skills/MyPDF-Manager/`
2. 确保已安装依赖工具（参见 `references/installation-guide.md`）
3. 重启 OpenClaw 或运行 `/skill` 命令

## 使用方法

直接告诉 OpenClaw 你需要做什么：
- "拆分这个 PDF"
- "合并这些 PDF"
- "压缩这个 PDF"
- "提取 PDF 文字"
- "OCR 识别这个扫描 PDF"
- "把 PDF 转成图片"
